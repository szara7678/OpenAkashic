## Level Map

### Starter
- 목표: 히라가나/가타카나, 자기소개, 아주 짧은 인사와 요청.
- 자료: NHK Easy Japanese, Minato starter courses.
- 루틴: 매일 10-15분 듣고 따라 말하기, 문자 읽기, 예문 3개 직접 바꾸기.

### A1
- 목표: 익숙한 일상 상황에서 짧은 문장으로 말하고 이해하기.
- 자료: Marugoto A1, JF Standard Can-do A1, Erin basic skits.
- 루틴: `can-do` 하나를 고르고, 단어 10개, 문형 2개, 짧은 대화 1개를 묶는다.

### A2
- 목표: 쇼핑, 이동, 일정, 가족, 취미 같은 생활 주제를 설명하기.
- 자료: Marugoto A2, Irodori Starter/Elementary, Erin basic to advanced.
- 루틴: 주제별 단어장을 만들고, 역할극 대화를 3턴 이상 작성한다.

### B1 Bridge
- 목표: 이유, 비교, 경험, 계획을 더 길게 설명하기.
- 자료: JF Standard Can-do B1 목표, Irodori 확장, 실제 기사/영상 일부.
- 루틴: 같은 주제로 말하기 요약, 읽기 요약, 내 의견 3문장을 만든다.

### JLPT Track
- 목표: 시험 대비용 독해/문법/청해 분리 학습.
- 자료: JLPT level summary를 기준으로 N5부터 N1까지 요구 수준을 확인한다.
- 주의: 공식 JLPT는 세부 단어/문법 전체 목록보다 레벨별 능력 설명을 기준으로 공개하므로, vocabulary list는 비공식 자료를 evidence로 쓸 때 출처와 한계를 표시한다.

## Vocabulary Structure
- `starter-vocab`: 문자, 숫자, 시간, 장소, 기본 동사.
- `daily-vocab`: 쇼핑, 교통, 식사, 건강, 약속.
- `work-vocab`: 메일, 일정, 요청, 보고, 회의.
- `media-vocab`: 뉴스, 날씨, 사건, 의견 표현.
- `kanji-basic`: 숫자, 날짜, 사람, 장소, 방향, 빈도 높은 생활 한자.
- `onomatopoeia-basic`: 일상 대화에 자주 나오는 의성어/의태어 대표 패턴.

## Grammar Structure
- Starter: `です`, `ます`, `は/が/を/に/で`, 지시어, 의문문.
- A1: 형용사, 동사의 현재/과거, `て`형 입문, 존재 표현.
- A2: 이유 `から/ので`, 희망 `たい`, 경험 `たことがある`, 비교, 조건 입문.
- B1 Bridge: 수동/사역, 추측, 양보, 인용, 복문 연결.

## Capsule Buckets
- `can-do`: 상황별로 무엇을 할 수 있는지 정의한다.
- `vocab`: 단어, 품사, 직접 만든 예문, 같이 쓰는 표현을 저장한다.
- `grammar`: 의미, 형태, 짧은 예문, 흔한 실수, 유사 표현 차이를 저장한다.
- `listening`: 짧은 음성/영상 자료 링크, 받아쓰기 포인트, 따라 말하기 문장을 저장한다.
- `reading`: 짧은 글 주제, 핵심 표현, 요약 문장, 내 의견 문장을 저장한다.
- `jlpt`: 레벨 목표, 자주 틀리는 유형, 연습 루틴, 체크리스트를 저장한다.

## OpenAkashic Use
- 난이도별 단어/문법은 `capsule` 또는 `reference`로 쪼개 저장한다.
- 학습자가 직접 겪은 성공/실패 루틴은 private note로 저장한 뒤 공개 가능하면 publication request를 보낸다.
- 공식 자료 링크는 evidence로 남기고, 실제 학습법은 공개 가능한 capsule로 정리한다.
- 단어 목록은 저작권과 출처를 확인하고, 대량 복제보다 대표 예시와 생성 규칙 중심으로 남긴다.

## Source Notes
- JLPT 공식 사이트는 N1-N5의 언어 지식, 독해, 청해 요구 수준을 설명한다.
- JF Standard는 일본어 사용 상황에서 무엇을 할 수 있는지 `Can-do`로 목표를 잡게 해 준다.
- Minato와 Marugoto는 Japan Foundation의 온라인 학습 흐름을 제공한다.
- Erin's Challenge는 영상, 표현, 문화 요소를 함께 학습하는 데 적합하다.
- Irodori는 일본 생활 장면 중심으로 학습 목표를 잡기 좋다.
- NHK Easy Japanese는 입문자의 듣기/표현 루틴에 쓰기 좋다.

## Reuse
일본어 관련 요청은 이 문서로 레벨을 먼저 정하고, 관련 capsule을 찾아 응답한다. 새 단어장이나 문법 정리는 `Japanese Learning Capsules`에 추가한다.
